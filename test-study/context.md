# Course Context: MATH 223 - Linear Algebra

## Test Format
- 5 computation problems (60% of grade)
- 2 conceptual/proof questions (40% of grade)
- 90 minutes, closed book
- Calculator NOT allowed

## Notation
- Matrices denoted with capital letters: A, B, C
- Vectors with lowercase bold or arrows: **v**, **w**
- Determinant written as det(A) or |A|
- Eigenvalues as lambda (λ), eigenvectors as **v**

## Professor's Emphasis
- Prefers cofactor expansion along row/column with most zeros
- Expects geometric interpretation alongside computation
- Past exams always include one "diagonalize or explain why not" problem
- Frequently tests: "what happens to det when you do [row operation]?"

## Past Exam Problems
1. Find the eigenvalues and eigenvectors of A = [[2, 1], [1, 2]]
2. Compute det([[1, 2, 3], [0, 4, 5], [1, 0, 6]]) using cofactor expansion
3. Is B = [[3, 0], [0, 3]] diagonalizable? If so, diagonalize it. If not, explain why.
4. If det(A) = 5, what is det(3A) for a 3x3 matrix? Explain your reasoning.
